# CLBV Website

Website tinh cho CLBV, duoc tao truc tiep tren GitHub ma khong can Git local.

## Cau truc

- `index.html`: Noi dung trang chu.
- `styles.css`: Giao dien responsive.
- `script.js`: Menu mobile va tuong tac form co ban.

## Chay website

Mo file `index.html` tren trinh duyet hoac bat GitHub Pages trong repository:

1. Vao **Settings** cua repo.
2. Chon **Pages**.
3. Tai **Build and deployment**, chon **Deploy from a branch**.
4. Chon branch `main` va folder `/root`.
5. Bam **Save**.

Sau khi GitHub Pages build xong, website se co URL dang:

`https://ainnhu.github.io/CLBV/`

## Tuy bien nhanh

- Doi email nhan dang ky trong `index.html`, tim `hello@example.com`.
- Cap nhat lich su kien trong section `#events`.
- Doi anh nen hero trong `styles.css`, tim `.hero-media`.
